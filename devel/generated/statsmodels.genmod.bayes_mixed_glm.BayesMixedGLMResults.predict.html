statsmodels.genmod.bayes_mixed_glm.BayesMixedGLMResults.predict¶

BayesMixedGLMResults.predict(exog=None, linear=False)[source]¶

Return predicted values for the mean structure.

Parameters:
  • exog (array-like) – The design matrix for the mean structure. If None, use the model’s design matrix.
  • linear (bool) – If True, returns the linear predictor, otherwise transform the linear predictor using the link function.
Returns:
Return type:

A one-dimensional array of fitted values.

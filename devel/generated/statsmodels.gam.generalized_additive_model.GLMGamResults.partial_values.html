statsmodels.gam.generalized_additive_model.GLMGamResults.partial_values¶

GLMGamResults.partial_values(smooth_index, include_constant=True)[source]¶

contribution of a smooth term to the linear prediction

Warning: This will be replaced by a predict method

Parameters:
  • smooth_index (int) – index of the smooth term within list of smooth terms
  • include_constant (bool) – If true, then the estimated intercept is added to the prediction and its standard errors. This avoids that the confidence interval has zero width at the imposed identification constraint, e.g. either at a reference point or at the mean.
Returns:

  • predicted (nd_array) – predicted value of linear term. This is not the expected response if the link function is not linear.
  • se_pred (nd_array) – standard error of linear prediction
